# Installation Guide

Welcome to the installation guide. This document will walk you through setting up the project on your local machine.

## Prerequisites

Before you begin, ensure you have the following installed:

- Python 3.8 or higher
- pip package manager
- Git (optional, for cloning)

## Quick Install

Install the required dependencies using pip:

```bash
pip install markdown pyyaml
```

## Verify Installation

Check that everything is working:

```bash
python3 --version
pip show markdown
```

## Next Steps

Once installed, proceed to the [Configuration Guide](configuration.html) to set up your project.
